# brightlocal.html CSV schema

## brightlocal-kpis.csv
Columns: `metric,value,subtitle,status`
Example: `Top 3 Rankings,5,Keywords ranking locally,good`

## brightlocal-rank-tracker.csv
Columns: `keyword,location,position,prev_position,change`
Example: `dtf printing,New Jersey,4,6,-2`

## brightlocal-competitors.csv
Columns: `keyword,competitor,position,url`
Example: `dtf printing,Example Prints,1,https://example.com`

## brightlocal-gbp.csv
Columns: `metric,value,share_pct`
Example: `Website clicks,120,42.0`

## brightlocal-actions.csv
Columns: `area,recommendation,priority`
Example: `GBP,Add new service photos,Medium`
