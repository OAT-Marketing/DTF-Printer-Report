# bing.html CSV schema

## bing-trend.csv
Columns: `month,impressions,clicks,ctr,position`
Example: `Apr 2026,46294,234,0.51,12.4`

## bing-queries.csv
Columns: `keyword,impressions,clicks,ctr,position`
Example: `dtf transfers,1200,30,2.5,8.1`

## bing-pages.csv
Columns: `page,impressions,clicks,ctr,position`
Example: `/collections/dtf-transfers,980,22,2.24,7.9`
