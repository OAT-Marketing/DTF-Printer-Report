# shopify.html CSV schema

## shopify-monthly.csv
Columns: `month,channel,orders,cvr`
Example: `Apr 2026,Google,859,9.42`

## shopify-orders-by-channel.csv
Columns: `channel,apr_orders,mar_orders,mom`
Example: `Google,859,763,12.6%`

## shopify-cvr-by-channel.csv
Columns: `channel,apr_cvr,mar_cvr,mom_delta`
Example: `Google,9.42%,8.99%,+0.43pp`
