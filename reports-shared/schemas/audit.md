# audit.html CSV schema

## audit-kpis.csv
Columns: `metric,value,subtitle,status`
Example: `Critical Issues,4,Needs priority cleanup,high`

## audit-issues.csv
Columns: `category,issue,count,severity,fix`
Example: `Titles,Duplicate title tags,12,High,Rewrite unique page titles`

## audit-actions.csv
Columns: `area,action,priority,owner,status`
Example: `Metadata,Rewrite duplicate titles,High,SEO,Open`
