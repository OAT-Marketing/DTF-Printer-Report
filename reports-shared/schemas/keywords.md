# keywords.html CSV schema

## keywords-kpis.csv
Columns: `metric,value,subtitle`
Example: `Tracked Clusters,6,Keyword groups monitored`

## keywords-monthly.csv
Columns: `keyword,cluster,month,impressions,clicks,position`
Example: `dtf transfers,DTF,Apr 2026,1200,42,8.4`

## keywords-clusters.csv
Columns: `cluster,keywords,impressions,clicks,avg_position,status`
Example: `DTF,20,5000,180,12.2,ranking`

## keywords-variants.csv
Columns: `seed,keyword,cluster,clicks,position`
Example: `dtf transfers,custom dtf transfers,DTF,20,9.8`
