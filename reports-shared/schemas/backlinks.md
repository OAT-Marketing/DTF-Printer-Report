# backlinks.html CSV schema

## backlinks-summary.csv
Columns: `metric,value,subtitle`
Example: `Referring Domains,45,Current Ahrefs snapshot`

## backlinks-new.csv
Columns: `date,referring_url,target_url,dr,type,status`
Example: `2026-04-15,https://example.com/article,https://dtfjersey.com/products/dtf-transfers,42,Content,Live`

## refdomains.csv
Columns: `domain,dr,domain_traffic,links,first_seen`
Example: `example.com,42,1500,3,2026-04-15`
